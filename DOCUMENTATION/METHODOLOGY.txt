## ⚙️ Methodology – Stack-Based Browser Navigation

The methodology of this project is based on simulating real browser navigation using the **Stack data structure**, which follows the **Last-In-First-Out (LIFO)** principle. Two stacks are used to manage browsing history: a Back Stack and a Forward Stack.

### 1. Data Structure Selection

* **Back Stack**: Stores previously visited web pages.
* **Forward Stack**: Stores pages that are navigated back from.
* **Current Page Variable**: Keeps track of the active webpage.

Stacks are chosen because they naturally support LIFO behavior, which matches browser history navigation.

---

### 2. Page Navigation Process

When a user visits a new page:

* The current page (if any) is pushed onto the Back Stack.
* The Forward Stack is cleared to maintain correct browser behavior.
* The new page becomes the Current Page.

---

### 3. Back Navigation Process

When the Back button is clicked:

* The system checks if the Back Stack is empty.
* If not empty:

  * The current page is pushed onto the Forward Stack.
  * The top page from the Back Stack becomes the Current Page.
* If empty, an error message is displayed indicating no history.

---

### 4. Forward Navigation Process

When the Forward button is clicked:

* The system checks if the Forward Stack is empty.
* If not empty:

  * The current page is pushed onto the Back Stack.
  * The top page from the Forward Stack becomes the Current Page.
* If empty, an error message is displayed.

---

### 5. Display Module

After every operation (Open, Back, Forward), the current page is displayed to reflect the updated browsing state.

---

### 6. Working Principle

The entire system works on:

* **LIFO (Stack Principle)**
* Controlled push and pop operations
* Real-world simulation of browser history management

---

### 🧠 Outcome

This methodology successfully demonstrates how stack data structures are used in real-world applications like web browsers to efficiently manage navigation history.
